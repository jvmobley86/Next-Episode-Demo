Next Episode

​
​

Next Episode is an app that was created with your favorite tv shows in mind. To store user profiles and tv shows, we used a Mongoose database. We used epguides API to run user searches and return tv show data. We also scraped the TV Guide website for posters to add to our show cards in the user dashboard.

​

Users are able to add and remove their favorite shows while we keep track of which episodes are up next.
